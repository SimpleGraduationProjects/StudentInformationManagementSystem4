## 技术选型

- Spring+SpringMVC+Mybatis+MySQL+easyUI

## 功能

- 增删查改、分页、模糊查询、权限管理、登录拦截

- 管理员列表（增加、查看、修改、删除、批量删除）
- 年级列表（增加、查看、修改、删除、批量删除）
- 班级列表（增加、查看、修改、删除、批量删除）
- 学生列表（修改）
